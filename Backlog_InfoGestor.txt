

Resumo do que o aplicativo deve fazer

1) Adicionar novos clientes
2) Sitema de OS com numeração e busca
3) Venda junto com Estoque para gereciamento de fluxo de caixa
4) Agenda com alerta para avisos de compromissos agendados com clientes
5) Um Dashboard simples
6) Um Sistema de contas a pagar e receber para uso pessoal
7) Sistema de autenticação de usuário
8) Software instalável
